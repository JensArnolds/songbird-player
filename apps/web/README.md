# apps/web

Next.js App Router web runtime.

- Source: `apps/web/src`
- Static assets: `apps/web/public`
- Dev/prod server wrapper: `apps/web/scripts/server.js`
- Drizzle config/schema: `apps/web/drizzle*`, `apps/web/src/server/db`

Common commands:

- `npm --prefix apps/web run dev`
- `npm --prefix apps/web run build`
- `npm --prefix apps/web run start`
